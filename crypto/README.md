# gryphCTF Cryptography Challenges

Created by: Mudit

# Overview
Welcome to the Cryptography challenges directory! These challenges are set in a medieval world, tasking you with intercepting royal decrees, decoding manuscripts, uncovering secrets, and more.

The challenges range from Very Easy to Very Hard, covering everything from ciphers to modern lattice-based attacks. While some may look like simple strings of text, others will require you to apply digital forensics techniques on images or write advanced scripts to recover hidden keys.
